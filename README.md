Go言語でorg-pomodoroが有効かを監視して、開始されたら録画を開始し、停止されたら録画を停止するジョブサーバを作る。

タスクタイトルをとる
emacsclient -e 'org-clock-heading'

ポモドーロがアクティブか調べる
emacsclient -e 'org-pomodoro-active-p'
